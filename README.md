# 🐺 狼人殺輔助器 Werewolf Assistant

線下狼人殺的網頁輔助工具。支援 **8～16 人**，提供 **語音引導** 與 **規則防呆機制**，
讓桌邊不需要實體法官也能順暢進行。

## ✨ 功能特色

- **自由配置**：自選遊玩人數（8–16）與要使用的角色，內建各人數推薦板子。
- **兩種模式**
  - 🎮 **遊玩模式**：建立 4 位數房號，其他人用手機掃 QR 或輸入房號加入。系統自動：
    抽號碼 → 私下發角色（狼人互相可見隊友）→ 語音引導每個夜晚 / 白天流程。
  - 🧑‍⚖️ **法官模式**：單機輔助。你就是法官，網頁負責語音提示、計時、記錄場上狀態與結算。
    完全在瀏覽器執行，可離線使用。
- **語音引導**：使用瀏覽器內建語音（zh-TW），免外部服務。可隨時開關。
- **防呆機制**（重點）
  - 🧪 **女巫第一晚不能自救**：若女巫第一晚被刀並嘗試用解藥救自己，系統會擋下並顯示「第一晚女巫不能自救！」。（可在設定改為「全程不能自救」或「皆可自救」）
  - 🧪 女巫同一晚預設不能同時用解藥與毒藥（可開關）。
  - 🛡️ **守衛不能連續兩晚守護同一人**。
  - 🛡️🧪 **同守同救**：守衛與女巫同時作用於同一人 → 該玩家視為死亡。
  - 🔫 **獵人 / 狼王被女巫毒死時無法開槍**。
  - 🤪 **白痴**被投票時翻牌不死、失去投票權；被刀或被毒則正常死亡。

## 🎭 支援角色

| 角色 | 陣營 | 能力 |
|------|------|------|
| 🐺 狼人 | 狼 | 每晚與同伴擊殺一人 |
| 👑 狼王 | 狼 | 出局時可開槍（被毒除外） |
| 👨‍🌾 平民 | 民 | 無技能，靠推理投票 |
| 🔮 預言家 | 神 | 每晚查驗一人好 / 壞 |
| 🧪 女巫 | 神 | 解藥、毒藥各一瓶 |
| 🔫 獵人 | 神 | 出局時可開槍（被毒除外） |
| 🛡️ 守衛 | 神 | 每晚守護一人免疫狼刀（不可連守） |
| 🤪 白痴 | 神 | 被票出翻牌不死，失去投票權 |

夜晚行動順序：**守衛 → 狼人 → 女巫 → 預言家**。
勝利條件可選 **屠邊**（神全滅或民全滅即狼勝）或 **屠城**（好人全部出局狼才勝）。

## 🚀 執行方式

```bash
npm install
npm start          # 啟動於 http://localhost:3000
```

- 房主 / 法官開啟首頁，設定人數、角色與規則後選擇模式。
- 遊玩模式下，其他玩家用手機開啟 `http://<你的IP>:3000/player.html` 或掃描房號 QR 碼加入
  （手機需與主機在同一區網）。

## 🗂️ 專案結構

```
server/
  index.js          Express + Socket.IO 伺服器、房間事件
  game/
    roles.js        角色定義、推薦板子、配置驗證
    engine.js       規則引擎與所有防呆判定（前後端共用）
    rooms.js        房間管理
public/
  index.html        首頁：設定與選模式
  host.html         遊玩模式主畫面（房號、語音引導）
  player.html       玩家手機端（抽號碼、看角色、夜晚行動）
  judge.html        法官模式（瀏覽器直接執行引擎）
  join.html         以房號加入的轉址頁
  js/               voice.js（語音）、ui.js（共用元件）
  css/styles.css
```

規則引擎 `engine.js` / `roles.js` 為純 ES 模組，後端與法官模式前端共用同一份邏輯，
確保兩種模式行為一致。
